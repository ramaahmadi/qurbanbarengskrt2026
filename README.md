# Qurban Bareng SKRT 2026

Website pendaftaran Qurban Bareng SKRT 2026

## Fitur

- Form pendaftaran qurban dengan pilihan hewan (Kambing/Domba)
- Input harga hewan custom
- Kirim data pendaftaran langsung ke WhatsApp admin
- UI responsive untuk desktop dan mobile
- Desain modern dengan warna pastel yang lembut

## Cara Menggunakan

1. Clone repository
2. Install dependencies: `npm install`
3. Jalankan development server: `npm run dev`
4. Buka http://localhost:5173

## Build untuk Production

```bash
npm run build
```

## Teknologi

- React 19
- Vite
- CSS3 dengan responsive design
